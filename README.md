## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/benetesla/Test-Fron-end-VFlows.git
```

Entre no diretório do projeto

```bash
  cd Test-Fron-end-VFlows
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
